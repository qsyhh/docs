1. [点击此处下载XTerminal](https://www.xterminal.cn)

2. 下载完成后打开点击新建服务器（+号）

![](../Img/SSH/XTerminal新建ssh.png)

3. 写上一个名字/ip地址/登录密码后创建即可

![](../Img/SSH/XTerminal绑定ssh.png)

 - 你的登录密码会在你的服务商控制台显示（不知道在哪里可以去百度搜下）

4. 创建完成点击连接

![](../Img/SSH/XTerminal连接ssh.png)
