# NapCatQQ相关

1. NapCatQQ 是基于 PC NTQQ 本体实现一套无头 Bot 框架。

2. NapCat名字寓意 瞌睡猫QQ，像睡着了一样在后台低占用运行的无需GUI界面的NTQQ

3. [NapCat官方文档](https://napneko.github.io/zh-CN/)

4. 本教程以 NapCat V1.8.4 为例
