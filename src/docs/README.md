## 系统

- [Windows安装教程](windows/)
  - Windows系统安装真寻Bot教程
  - 需准备2核(CPU核心)4G(RAM内存)以上服务器
  - 版本要求win10+(2016+建议最新)
  - 请使用 AMD（Advanced Micro Devices）架构

- [Linux安装教程](linux/)
  - Linux(Ubuntu)发行版安装真寻Bot教程
  - 需准备2核(CPU核心)2G(RAM内存)以上服务器
  - 版本要求(Ubuntu 22)
  - 请使用 AMD（Advanced Micro Devices）架构
